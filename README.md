**Мета роботи**

Створити глобальну змінну, яка має 4 нижні індекси 3 різних типів (числа, дати, рядки, списки тощо). Використати як значення для вузлів обидва типи списків. Показати використання щонайменше 4 вбудованих функцій для роботи з масивами. Усі параметри повинні зчитуватися з пристрою введення. Глобальні змінні та функції повинні мати сенс.

2.	Результат виконання

Ілюстрація роботи глобальної змінної з 4 підскриптами та використанням вбудованих функцій для роботи з масивами
**($ORDER, $GET, $DATA, $QLENGTH)** 

<img src="img\2.png" alt="">

Програма зчитує дані про книгу з клавіатури, зберігає їх у глобальній змінній ^Books,
після чого демонструє перевірку, читання та ітерацію по елементах глобального масиву.
У завершенні виводиться повний список усіх книг, які були занесені до бібліотеки.

Функція	Призначення
**$DATA()**     -	Перевіряє, чи існує вузол глобала
**$GET()**      -	Безпечно отримує значення вузла глобала
**$ORDER()**    - Виконує ітерацію по індексах багатовимірного масиву
**$QLENGTH()**  -	Визначає кількість підскриптів (глибину вузла глобалу)

Додаткові функції
**$QSUBSCRIPT()**	- Отримує значення конкретного підскрипта вузла
**$QUERY()**	    - Послідовний обхід піддерева багатовимірного масиву

**Висновок**
Програма демонструє:
•	зберігання даних у глобальній змінній з 4 підскриптами різних типів,
•	ітерацію, читання та аналіз структури за допомогою **$ORDER, $GET, $DATA, $QLENGTH,**
•	зручне відображення повного списку книг із глобалу.
Поставлена мета - реалізувати роботу багатовимірного глобалу та застосувати вбудовані функції ObjectScript успішно досягнута.

**Мета роботи:**

створити вебінтерфейс та програмні сервіси для перегляду та редагування студентів у базі InterSystems IRIS.

**Завдання**

Використати CSP для створення веб-сторінок, які працюють з об’єктами класу з лабораторної №4. <br>
Вивести таблицю всіх елементів класу за допомогою прямого доступу до БД та додати кнопки: <br>
Додати студента <br>
Редагувати студента <br>
Видалити студента <br>
Створити RESTful сервіс та клієнт для створення, зміни і видалення об'єктів. <br>
Створити SOAP сервіс та клієнт для створення, зміни і видалення об'єктів. <br>

RESTGetAll() <br>
Відображає список студентів у форматі JSON. Це демонструє, що REST-сервіс успішно читає дані з бази. <br>

RESTCreateOne() <br>
Створює нового студента через REST-запит. У відповіді повертається {"status":"ok","ID":33} — тобто студент доданий. <br>

SOAP Demo() → CreateStudent <br>
Відправляється SOAP-запит для створення студента. Отримано XML-відповідь <CreateStudentResponse> — операція виконана.<br>

SOAP Demo() → GetStudent(ID=1) <br>
SOAP-клієнт запитує дані студента з ID = 1. У відповіді <GetStudentResult>false</GetStudentResult> — тобто студент з таким ID не знайдений або був видалений.<br>


**Висновок**

У роботі я створив повноцінний вебінтерфейс для керування об’єктами Online.Student.
CSP забезпечує роботу користувача через браузер.
REST та SOAP дозволяють іншим програмам звертатися до бази для створення, редагування та видалення студентів.

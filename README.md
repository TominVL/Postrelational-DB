**Мета роботи:**

Створити класи бази даних на основі діаграми класів із Практичної роботи №3.
Необхідно створити щонайменше два об’єкти кожного класу, встановити всі їхні властивості з урахуванням обмежень (унікальність, обов’язковість тощо) та зберегти об’єкти в базі даних.
Значення властивостей повинні бути логічними й узгодженими між собою.
